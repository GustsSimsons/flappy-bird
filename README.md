# Flappy Bird Clone

A 2D arcade clone of the original Flappy Bird game. This project was built to practice game physics, collision detection, and state management.

## 🚀 Features

* **Physics-Based Movement:** Realistic gravity and upward flap impulses.
* **Procedural Obstacles:** Randomly generated pipe heights with consistent gap sizing.
* **Collision Detection:** Precise bounding box tracking for pipes, ground, and ceiling.
* **Score System:** Point increments every time a pipe pair is successfully cleared.
